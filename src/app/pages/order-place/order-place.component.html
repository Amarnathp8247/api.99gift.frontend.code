PUT /gift/order-create-corporate

Create Order

Create a new order for a corporate gift. The request payload must be encrypted using CryptoJS. Token is required and must be passed in the header.

Note: The actual payload should be encrypted and sent as a "data" field in the request.

Request Attributes

Attribute Type Required Description Sample
authcode string Yes Enter this authcode in Payload "XXXX"
customerName string Yes The name of the customer making the purchase "Joy Mathur"
emailID string Yes The email address of the customer "joymathur@gmail.com"
mobileNo string Yes The mobile number of the customer "0123456789"
walletPayment boolean Yes Indicates if the payment is made using a wallet false
productId integer Yes The unique identifier of the product being purchased 691
device_type integer Yes The type of device used for the purchase (0 - Android, 1 - Mobile) 0
denominations array Yes An array of objects containing details of the denominations being purchased See below
corp_discount float No Discount value for the corporate product 3.5
skuID string Yes The unique ID for the product "OFFXXXGLENW"

Denominations Object
